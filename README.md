# 🚀 Collaborative Code Editor

A real-time collaborative code editor built using:

- React
- Node.js
- Express
- Socket.io
- Monaco Editor
- Judge0 API
- **Arcee AI Trinity Large** (for intelligent chatbot)

## Features

- Multi-user rooms
- Real-time code sync
- Live chat
- Version history
- Multi-language execution
- Shared output system
- **🤖 AI-Powered Code Assistant** (Trinity Large)

## Tech Stack

Frontend: React + Vite  
Backend: Node + Express + Socket.io  
Code Execution: Judge0 API  
AI Assistant: Arcee AI Trinity Large (via Replicate API)
